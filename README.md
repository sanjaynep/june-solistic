Getting started

Prerequisites
- A modern browser.
- (Optional) Node.js and a static server for local dev, or Python for a quick local server.

Quick preview (no install)
- Open `index.html` in your browser.

Run a simple local server
- Using Python 3:
  - python -m http.server 8000
  - Open http://localhost:8000
- Using Node (http-server):
  - npm install -g http-server
  - http-server -c-1
  - Open the printed local URL

Project structure
- index.html — main page
- assets/ — images, icons, fonts, and other static assets
- css/ — optional stylesheets
- js/ — optional JavaScript files
- README.md — this file

How to customize
- Update content directly in `index.html`.
- Place images in `assets/` and reference them from HTML.
- Add or update styles in `css/` and scripts in `js/`.

Deployment
- GitHub Pages:
  - Push to `main` branch and enable GitHub Pages in repository settings (source: main / root).
  - Or push to `gh-pages` branch and use that as the Pages source.
- Any static host (Netlify, Vercel, Firebase Hosting, Surge).

Accessibility checklist (suggested)
- Use semantic HTML sections (header, nav, main, footer).
- Add alt text for all images.
- Ensure color contrast and keyboard navigability.
- Include a skip-to-content link for screen-reader users.
